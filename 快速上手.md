# 🛒 闲鱼数据调研工具 - 快速上手指南

## 你是什么都不会的大学生？没关系，按这个步骤来：

---

## 第一步：安装 Python（如果还没有）

1. 打开浏览器，访问 https://www.python.org/downloads/
2. 点击黄色大按钮下载 Python
3. 安装时 **一定勾选 "Add Python to PATH"**（重要！）
4. 一路 Next 完成安装

验证安装：按 `Win+R`，输入 `cmd`，回车，输入 `python --version`，看到版本号就成功了。

---

## 第二步：下载本项目

把整个 `xianyu-tool` 文件夹复制到你电脑上任意位置（如桌面）。

---

## 第三步：安装依赖

1. 打开 `xianyu-tool` 文件夹
2. 在地址栏输入 `cmd` 并回车（打开命令行）
3. 输入以下命令：

```bash
pip install -r requirements.txt
```

4. 安装浏览器驱动：

```bash
python -m playwright install chromium
```

---

## 第四步：启动软件

在同一个命令行窗口输入：

```bash
python main.py
```

软件界面就会弹出来！

---

## 第五步：使用软件

1. **输入关键词** → 比如"蓝牙耳机"、"机械键盘"、"考研资料"
2. **设置数量** → 建议 30-50 条，最多 100 条
3. **点击开始采集** → 会自动弹出浏览器窗口
4. **扫码登录** → 首次使用需要扫码登录闲鱼（之后会自动记住）
5. **等待采集** → 每采集一条商品会间隔几秒（防止被封号）
6. **查看结果** → 切换到「数据预览」标签页
7. **导出 Excel** → 点击按钮导出，包含三个工作表

---

## 第六步（可选）：打包成独立 .exe

如果不想每次都用命令行启动，可以打包成 .exe：

1. 双击 `build.bat`
2. 等待几分钟
3. 完成后在 `dist` 文件夹找到 `闲鱼数据调研工具.exe`
4. 双击就能运行！

---

## ⚠ 重要提醒

| 能做的 | 不能做的 |
|--------|----------|
| 小规模采集（几十条） | 大规模批量爬取（几百上千条） |
| 个人学习研究 | 商业用途 |
| 低频使用 | 短时间内反复采集 |
| 手动导出分析 | 传播爬取的数据 |

## 文件说明

```
xianyu-tool/
├── main.py              ← 主程序入口
├── build.bat            ← Windows 打包脚本（双击运行）
├── requirements.txt     ← 依赖清单
├── 快速上手.md           ← 你正在看的文件
├── core/                ← 核心模块
│   ├── config.py        ← 配置管理（防封策略等）
│   ├── crawler.py       ← 爬虫引擎
│   ├── database.py      ← 数据库
│   ├── analyzer.py      ← 文案分析
│   └── exporter.py      ← Excel 导出
├── gui/                 ← 图形界面
│   └── main_window.py   ← 主窗口
├── data/                ← 数据存储（自动创建）
├── exports/             ← 导出文件（自动创建）
└── images/              ← 下载的图片（自动创建）
```

---

## 遇到问题？

1. **启动报错 "No module named xxx"** → 运行 `pip install -r requirements.txt`
2. **浏览器打不开** → 运行 `python -m playwright install chromium`
3. **采集不到数据** → 检查是否已登录闲鱼、闲鱼页面结构可能更新
4. **被杀毒软件拦截** → 添加信任（Python 和 Playwright 都是正规软件）
